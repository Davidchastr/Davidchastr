## Hi there 👋
Я провёл тест-анализ и спроектировал проверки на часть требований к приложению «Яндекс Маршруты». Для всего приложения я выполнил следующие задачи:

Проанализировал и декомпозировал требования.

Составил диаграмму связей, чтобы понять, какая функциональность уже реализована в приложении.

Для одного вида транспорта я выполнил следующие задачи:

Составил блок-схему, чтобы понять, как работает логика приложения для выбранного вида транспорта.

Подготовил таблицу с контрольными точками (КЭ), чтобы получить тестовые данные для будущих тест-кейсов.

Разработал несколько тест-кейсов на основе подготовленных тестовых данных.

В результате были разработаны проверки, которые позволят убедиться в соответствии функциональности приложения требованиям. Это поможет обеспечить высокое качество продукта и удовлетворение потребностей пользователей.
